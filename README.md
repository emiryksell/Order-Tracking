**RestoPOS** is a browser-based Point of Sale (POS) simulation designed for restaurants and cafes, featuring a sleek **Glassmorphism** UI. It manages tables, orders, and payments entirely in the browser using **LocalStorage**, requiring no backend or database setup.

## üåü Features

* **Table Management:** 12-table grid view with instant status tracking. Occupied tables feature a glowing "Pulse" animation.
* **Quick Order System:** Categorized menu items and a smart **Search Bar** with instant filtering for fast operation.
* **Payment Options:** Process payments via **Cash** or **Credit Card** and track them separately in reports.
* **Receipt Simulation:** View a realistic thermal receipt preview before completing the payment.
* **Advanced Reporting:**
    * Daily Total Revenue & Order Count
    * Cash vs. Credit Card Breakdown
    * Best-Selling Product Analysis
    * **Excel Export:** Download daily sales reports as a `.csv` file with one click.
* **Persistent Data:** All data (orders, sales history) remains saved even after refreshing the page (via LocalStorage).
* **Modern UI:** Stylish background, blurred glass effects (Glassmorphism), and custom scrollbar design.
